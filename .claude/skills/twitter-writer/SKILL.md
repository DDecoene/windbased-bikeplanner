# Tweet Writer Skill — rgwnd

## Overview

Deze skill helpt je virale, overtuigende social media posts te schrijven voor rgwnd, geoptimaliseerd voor het X-algoritme. Posts zijn altijd in het Nederlands, weergestuurd, en verwijzen naar Vlaamse provincies en het fietsknooppuntennetwerk.

**rgwnd** (rugwind) is een fietsrouteplanner die lusroutes optimaliseert voor rugwind. Gratis te gebruiken op rgwnd.app. Draait op het Belgische fietsknooppuntennetwerk.

## Het X-Algoritme (2026)

### Engagement Hiërarchie (Meest naar Minst Waardevol)
1. **Replies** — Zwaarst gewogen signaal
2. **Quote tweets** — Hoge waarde
3. **Bookmarks** — Sterk waardesignaal
4. **Retweets** — Amplificatie
5. **Likes** — Basissignaal

### Wat te VERMIJDEN
- **Externe links**: Zwaar bestraft (rgwnd.app NIET linken, alleen benoemen)
- **Hashtags**: Geen — zien er spammy uit
- **Vragen om engagement**: "Like en RT" schaadt bereik
- **Generieke content**: Geen onderscheid = geen bereik

## Hook Formules voor Fietsen

### De Weerhook
```
"[Temperatuur]°C en [wind] km/u uit het [richting] in [stad] — [conclusie over fietsen]"
"Wind uit het [richting] vandaag? [Hoe rgwnd dit voordeel maakt]"
```

### De Beste-Dag Hook
```
"Beste fietsdag deze week in [provincie]? [Dag]. [Specifieke cijfers]."
"[Dag] wordt DE dag: [temp], [wind], [regen]. Plan nu op rgwnd."
```

### De Seizoenshook
```
"Het wielerseizoen is geopend in [provincie]!"
"Eerste lentedag in [stad] — [specifieke condities]."
"[Wielerevenement] inspireert? Plan je eigen route in [provincie]."
```

### De Uitdagingshook
```
"Met rugwind trap je makkelijk [X] km verder. Plan een [Y] km lus op rgwnd."
"Wind mee = minder moeite. Hoeveel kilometer durf jij vandaag?"
```

### De Contraire Hook
```
"Regen in [stad]? Perfect moment om je weekendroute te plannen op rgwnd."
"Vandaag niet ideaal, maar [dag] wordt het wel: [cijfers]."
```

### De Feature Hook
```
"Wist je dat rgwnd tot 16 dagen vooruit kan plannen?"
"GPX exporteren voor Garmin of Wahoo? rgwnd doet het in één klik."
"rgwnd gebruikt het volledige Belgische knooppuntennetwerk."
```

## Post Formaten

### Formaat 1: Weer + Actie
```
[Weerfeit over stad/provincie]. [Conclusie/advies]. [rgwnd vermelding].
```
Voorbeeld: "18°C en amper wind in Gent vandaag. Perfecte condities voor een knooppuntenroute. Laat rgwnd je de ideale lus plannen — met rugwind als bonus."

### Formaat 2: Beste Dag Kroning
```
Beste fietsdag deze week in [provincie]? [Dag]. [Temp]°C, [wind] km/u, [regen]% regen. Plan je route op rgwnd.
```

### Formaat 3: Contrair Weer
```
[Slecht weer feit]. [Positieve draai]. [Toekomstige dag + condities].
```

### Formaat 4: Feature + Weer
```
[Feature beschrijving]. [Koppeling aan huidige condities in provincie].
```

### Formaat 5: Uitdaging
```
[Rugwind voordeel]. [Uitdaging voorstellen]. [rgwnd als enabler].
```

## Copywriting Frameworks

### PAS (Probleem → Agitatie → Oplossing)
```
[Probleem]: Altijd tegenwind op de terugweg?
[Agitatie]: Je eindigt je rit uitgeput terwijl het niet hoeft.
[Oplossing]: rgwnd plant je lus zodat je altijd wind mee hebt op de zware stukken.
```

### BAB (Before → After → Bridge)
```
[Before]: Vroeger plande ik routes en hoopte ik op goed weer.
[After]: Nu check ik rgwnd en weet ik exact wanneer en waar ik moet fietsen.
[Bridge]: Het verschil? Winddata + slimme routeplanning.
```

## Vlaamse Provincies — Context

Gebruik lokale referenties per provincie:
- **West-Vlaanderen** (Brugge): kust, polders, vlak, westenwind
- **Oost-Vlaanderen** (Gent): Schelde, studentenstad, levendige fietsscene
- **Antwerpen** (Antwerpen): Kempen, haven, stedelijk fietsen
- **Vlaams-Brabant** (Leuven): Hageland, heuvels, universiteitsstad
- **Limburg** (Hasselt): fietsparadijs, mijnstreek, fietsknooppunten-pionier
